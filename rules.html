Rules

You will read these because you will play by these

ABOUT

  • This contest is internal to the Students of IIT Madras BSc.
  • The theme is to build a cross-platform mobile app that benefits the students of IIT Madras BSc Degree
  • Participation can be either as an individual or as a team with a maximum of 4 members. We encourage you to form a team.
  • Submissions will be shortlisted and reviewed by a Judge Panel.
  • All submissions will be evaluated for plagiarism.
  • We strongly recommend Flutter to build your app as it is easy to pick up and has a fast development cycle that aligns with the timeline of the event. However, we shall entertain submissions built on other platforms as well.
  • Any and all decisions made by the organizing committee and judges will be final.
  • The submissions are property of IIT Madras.

ASSUMPTIONS

  • The application can at maximum assume that the following data will be available for any user : { student email, firstname, lastname, house, role, exam_city }
  • If required, A JSON File mapping course videos to their Youtube Links will be shared down the line.
  • Any additional details like CGPA,CCC, Courses Enrolled, Language Preferences have to be collected from the user on a voluntary basis.
  • Participants can choose to use the IIT Madras BSc Brandkit provided here.

SYSTEM REQUIREMENTS

  • At least one team member must meet the minimum system requirement to run Flutter and an emulator.
  • If you don't meet the system requirements but want to participate in Live Sessions purely for the sake of learning flutter, you can use the online IDE available at flutlab.io

Operating System

  • Windows: 7 or higher (64-bit)
  • MacOS X 10.10 (Yosemite) or higher
  • Linux : GNOME or KDE Desktop, 64-bit distro capable of running 32-bit applications with GNU C Library

Hardware

  • RAM: (Minimum) 4GB, (Recommended) 8GB.
  • Storage: (Minimum) 2GB free space. (Recommended) 4GB free space.
  • GPU: (Recommended) 2GB Integrated/Dedicated Graphics Memory

PRESENTATION

  • Participants will have to submit a presentation detailing their idea. Use the template provided here.
  • Submit your presentation here.
  • Deadline for Presentations : 23rd December 2021
